# Online Course Platform

A clean and scalable online course platform built with Django.

## Features

- Custom user authentication system with two roles (Admin/Regular User)
- Course management system with categories
- Favorites system
- Manual enrollment via WhatsApp contact
- Admin dashboard for complete control
- Modern, responsive design with dark mode
- Clean architecture with service layer pattern

## Requirements

- Python 3.11+
- PostgreSQL
- Django 4.2+
